FastAPI Demo Application

This is a simple RESTful API built with FastAPI and deployed on Firebase.

API Endpoints

GET /

Returns a welcome message.

GET /items

Returns all items in the database.

GET /items/{item_id}

Returns a specific item by ID.

POST /items

Creates a new item in the database.

DELETE /items/{item_id}

Deletes an item from the database.

Documentation

Full API documentation is available at /docs or /redoc endpoints.
